Projects

Currently working on uploading all of my older projects, but here are the recent ones!

fpir aka FASTX Parser In Rust - A fast Rust-based CLI tool for parsing FASTA/FASTQ files and computing basic sequence statistics. Needleman-Wunsch Algorithm Visualizer - A website that visualizes the Needleman-Wunsch algorithm for DNA regulation. React/Typescript/Tailwind. Long Read RNA Sequence Project - This project aims to analyze long-read RNA sequencing data derived from mouse brown adipose tissue to identify and quantify known and novel transcript isoforms using a basic bio-informatics data analysis pipeline: QC → alignment → correction → collapsing → quantifying → analysis. Various Game Dev projects - Was on the board of directors for gamedev and worked on some stuff!
